# CHIP-8-Emulator
A simple CHIP-8 emulator written in C++ using SDL3 for graphics and input handling. CHIP-8 is an interpreted programming language from the late 1970s.




# Features and Specifications
- Fully implemented CHIP-8 instruction set via [Cowgod's technical specs](http://devernay.free.fr/hacks/chip8/C8TECH10.HTM)

- Passes opcode test in the [Timendus test suite](https://github.com/Timendus/chip8-test-suite) for all 35 opcode instructions

- 4KB of memory

- 64x32 monochrome display

- 16-key hexadecimal keypad
